# Cognitive-API

## Develop Stack
* php(7.0)
* mysql

## Installation
* [composer]
```
* sudo yum install -y php70w php70w-gd php70w-mbstring php70w-fpm
* composer require "pear/http_request2"
* composer require "intervention/image"
```

## Execution
* composer install
* php artisan serve




[composer]:https://getcomposer.org/download/
